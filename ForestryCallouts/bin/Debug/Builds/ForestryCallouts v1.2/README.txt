In the "ForetryCallouts" download go to plugins/lspdfr drag both "ForetryCallouts.dll" and "ForetryCallouts.ini" both into
you're gta5/plugins/lpsdfr folder.

Load up the game and you should see ForetryCallouts has loaded notification if done correctly.
